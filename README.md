# Paper That Remains

Desktop-responsive implementation of the supplied Figma start screen. The artwork
scales proportionally with comfortable margins, and its Start area links to a
separate second-page route.

The homepage is assembled from Figma nodes `15:20`, `15:21`, `15:224`,
`15:225`, `15:226`, `15:212`, and `15:24`. Its background enters first, eight
paper fragments bounce upward, and three fragment clicks reveal the grouped
“Paper”, “That”, and layered “Remains” artwork before Start appears.

The introduction page follows Figma node `2:6`: the people enter first, followed
by the two text descriptions in sequence.

The second-page first stage uses Figma nodes `17:430`, `17:431`, `17:434`,
`17:438`, `17:445`, `17:446`, and `17:451`. Four character groups enter in
sequence, then five hidden paper regions reveal a shadow on hover and remain red
after they are clicked.

After all five paper regions are selected, “Where is Paper?” fades away and the
completion description appears at the exact Figma coordinates before navigation
unlocks.

The completion description then clears, the four character layers dissolve in
sequence, and the red paper objects resize into the Figma `31:488` arrangement.
The final description fades in after the objects settle.

The red arrow on the second page links to Figma node `2:9`. Its environmental
scene enters first and the description is revealed afterward.

The third-page arrow links to the layered Figma artwork around node `15:9`.
Four turquoise ovals enter in order, the upper copy types in, and monochrome
paper fragments appear. Hovering every fragment changes it to the matching red
version and unlocks the lower typewritten copy.

Pages three through five provide matching lower-left back navigation. Figma node
`2:18` is the final collage; its lower-right return arrow restarts the experience
from the first page.

Descriptions on the second, third, and fourth pages use sequenced fades. The
final collage includes pointer-driven fragment echoes. Moving across paper
fragments scatters and rotates their silhouettes before they settle back into the
original composition.

Open `index.html` directly, or serve the directory locally:

```sh
python3 -m http.server 4173
```
